# 🔊 SoundMaster: 음향기기 통합 관리 시스템

구글 폼즈, 스프레드시트, 그리고 커스텀 대시보드를 결합한 **프리미엄 장비 관리 솔루션**입니다.

## 📁 구성 요소
1.  **Google Forms**: 장비 등록 및 상태 변경 입력용 인터페이스
2.  **Google Sheets**: 데이터 저장소 및 이력 관리
3.  **GAS (Apps Script)**: 시트 데이터를 대시보드로 전달하는 브릿지 API
4.  **Premium Dashboard**: 실시간 현황 파악 및 인벤토리 관리를 위한 고성능 UI

## 🚀 설정 가이드 (Setup Guide)

### 1단계: 구글 폼 및 시트 준비
1.  **구글 폼 생성**: 다음 항목을 포함하여 만드세요 (드롭다운 추천).
    *   **장비명** (단수 텍스트)
    *   **카테고리** (드롭다운: 스피커, 마이크, 믹서, 케이블, 기타)
    *   **시리얼 번호** (단수 텍스트)
    *   **상태** (드롭다운: 가용, 대여중, 수리중, 분실)
    *   **위치** (드롭다운 필수): 지층 리바이벌 성전, 1층 유치부실, 1층 다윗의 장막, 1층 새가족1실, 1층 새가족2실, 2층 웨일즈성전, 2층 아주사성전, 2층 유아예배실, 2층 영아예배실, 3층 본당, 6층 존웨슬리홀 등
    *   **사용자** (단수 텍스트): 실사용자 이름
    *   **사용 목적** (단수 텍스트): 예) 주일 대예배, 금요철야 등
    *   **사용 부서** (드롭다운): 예배팀, 방송팀, 유치부, 유아부, 영아부, 청년부, 관리팀, 기타
    *   **사용 날짜** (날짜): 대여 또는 사용 시점
2.  **응답 시트 연결**: 폼 응답을 스프레드시트에 저장하도록 설정하세요.

### 2단계: 앱 스크립트(GAS) 설정
1.  스프레드시트에서 `확장 프로그램 > Apps Script` 클릭.
2.  제공된 `appsscript.gs` 파일의 내용을 모두 복사하여 붙여넣기.
3.  상단 `배포 > 새 배포` 클릭.
4.  유형 선택: `웹 앱`
    *   설명: `SoundMaster API`
    *   액세스할 수 있는 사용자: `모든 사용자` (Anonymous 접근 허용 필요)

### 3단계: 대시보드 연결
1.  `script.js` 파일을 열어 상단의 변수를 수정하세요.
    *   `GOOGLE_FORM_URL`: 생성한 구글 폼 URL 입력.
    *   `GOOGLE_SHEET_API`: 구글 폼에 연동된 스프레드시트의 링크 입력.
2.  `index.html`을 실행하면 실시간 데이터가 연동된 대시보드를 확인할 수 있습니다.

### 4단계: 구글 사이트(Google Sites) 임베드
1. 파일 목록 중에 **`google_sites_embed.html`** 파일이 추가되었습니다. 이 파일은 `index.html`, `style.css`, `script.js` 세 개를 구글 사이트용으로 **하나로 합친 최종 통합본**입니다.
2. `google_sites_embed.html` 파일을 메모장이나 텍스트 에디터로 열고, 안의 모든 소스코드를 복사(Ctrl+A, Ctrl+C) 합니다.
3. 구글 사이트(Google Sites) 편집 화면으로 이동합니다.
4. 우측 메뉴에서 `삽입 > 코드 삽입(Embed) > 코드 임베드(Embed code)`를 선택합니다.
5. 복사한 전체 코드를 붙여넣기(Ctrl+V)하고 `다음` -> `삽입`을 클릭합니다.
6. 대시보드 창 크기에 맞게 임베드 위젯의 너비와 높이를 적절히 조절합니다.
7. 우측 상단의 `게시`를 눌러 팀원들에게 공유하세요!

*   **프리미엄 디자인**: Glassmorphism 및 현대적인 타이포그래피 적용.
